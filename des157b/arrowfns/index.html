Sally's Function

This function intakes an array of nouns/subjects, and outputs an array of sentences that says "${noun} is cool!" - this function was inspired from many of Bill's examples that use either cheese or llama as the subject!

Named function

            

            function aVariable(nounsArr){
                return nounsArr.map(function(noun) (
                    `${noun} is cool!`
                ))
            }

Function Expression

            

            const aVariable = function(nounsArr){
                return nounsArr.map(function(noun) (
                    `${noun} is cool!`
                ))
            }

Arrow Function

            

            const aVariable = (nounsArr)=>{
                return nounsArr.map(noun => (
                    `${noun} is cool!`
                ))
            }
